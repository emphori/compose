# Stitch

A drop in extension for Promises, adding oodles of functional goodness through
composition and taking error handling to an entirely different level.

## Getting started

```sh
# Install the dependencies
npm ci
```

## Building

```sh
# Build the package
npm run build

# Or, watch for changes
npm run build -- -w
```

## Testing

```sh
# Run the tests
npm test
```
